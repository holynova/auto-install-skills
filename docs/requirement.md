## 需求
自动安装agent skills到这台电脑

1. 扫描这台电脑上有多少支持claude code skills或者agent skills的软件, 比如claud code, gemini cli, antigravity, cursor, windsurf等
2. 找到扫描出来的软件存放skills文件的目录
3. 从几个知名的github skills项目中下载skills, 比如
- https://github.com/anthropics/skills
- https://github.com/ComposioHQ/awesome-claude-skills
- https://github.com/obra/superpowers
4. 下载的skills文件复制到扫描出来的软件存放skills文件的目录

5. 用户可以选择是否安装
6. 用户可以选择软件范围
7. 自动更新功能, 重新拉取github上的skills文件, 再运行一遍, 重名的skills就覆盖
